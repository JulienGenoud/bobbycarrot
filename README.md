EPIC Epitech BOBBY CARROT
================================

* Using Three.js
* Minified version comming...
* play at http://nodejs-bobbycarrot.rhcloud.com/
* ~~old php mirror https://php-bobbycarrot.rhcloud.com/~~

### Features

* Move with arrow keys
* load objects
* collisions
* level (1+2)
